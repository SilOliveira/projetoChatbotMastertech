Projeto escolhido para o módulo 01 do curso:

Opção 02: Implementação de Chatbot para Atendimento automatizado

Como fizemos?

Foi um imenso prazer realizar e dar vida a este projeto, optamos por desenvolver um chatbot para a empresa VShape-Pro.
A nossa experiência junto a essa tarefa foi excepcional, utilizamos a plataforma da Dialogflow fornecida pela Google Cloud, do qual pegamos o código gerado para chatbot e inserimos na página do site VShape-Pro. Além disso, fizemos um outro exemplo aplicando um bot via whatsapp do qual inserimos uma API via Google Cloud para realizar os testes.
